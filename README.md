# Community-Yad2 - Community Ads Platform

Community-Yad2 allows users to create, view, and manage community posts, simulating a local marketplace.

## Project Goal
Showcases frontend development skills and CRUD operations using React.

## Key Features
- Create, edit, and delete posts: Rent and sell items
- View all community posts
- Responsive and user-friendly interface

## Technologies
React, JavaScript, Firebase / Local Storage

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Demo / Screenshots
https://community-yad2.vercel.app/

## Challenges & Learning
- Handling complex state across components
- Working with APIs and dynamic data
- Building an friendly user interface
